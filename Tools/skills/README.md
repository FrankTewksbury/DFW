# DFW Global Skills Index

> **What this is:** The index of all global skills in the DFW skills library.
> Each entry includes the skill name, its situation trigger, and the file path.
> Agents scan this index when facing a non-trivial task to find relevant skills
> before proceeding.
>
> **Scope:** Global skills only. Project-local skills are not indexed here.
>
> **Maintenance:** Update this index whenever a global skill is added, promoted,
> or deprecated. This is a registration obligation for every skill file in this directory.
